# 🔐 Login Form Project

A simple **Login Form** built using **HTML, CSS, and JavaScript**.
This project demonstrates basic **clean UI design**, and **user input handling** using JavaScript.

---

## 🛠️ Built With

* 🌐 **HTML5** – Structure of the login form
* 🎨 **CSS3** – Styling and layout
* ⚡ **JavaScript** – Form input checking

---

## ✨ Features

* User-friendly login interface
* Login button activation only when all fields are filled.
* Button text changes to “Logging In…” on click.
* Welcome alert message after login.
* Input fields reset automatically after login.
* Clean and modern UI
